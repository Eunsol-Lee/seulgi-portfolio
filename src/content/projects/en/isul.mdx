## Challenge

Package design for ISUL Cosmetics' holiday gift set. The core objective was to create a sensory unboxing experience from the moment the recipient opens the box through the process of discovering each product.

## Process

A thick hard box structure with magnetic closure was engineered. Inside, osmanthus-scented paper straps secure each product, and a pocket on the inner lid holds a postcard for handwritten messages. The exterior features silk lamination for a soft-touch finish.

## Result

Holiday gift set sales increased 230% year-over-year. 91% of purchasers reported that recipients found the packaging impressive, and the same structural design was re-commissioned for the following spring season.
